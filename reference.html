Reference page for all server calls 

POST Methods:
createNote(String name, int userID, int apartmentID, String noteName, text note)
	*creates note in the notes table

createEvent(String date, String name, int userID,int apartmentID)
	*creates event with data sent in post

requestToJoinApartment(int userId,int ApartmentID)
	*autojoins to apartment for now

createGroupMessage(int senderID,int apartmentID,String MSGText)
	*creates event with data sent in post

createUserSEC(String username,String password,String email)
	*creates user with POST method to be more secure than GET method 

deleteEvent(int id)
	*deletes event

createApartment(int owner,String AptName,String address)
	*creates a new apartment, adds the user to the apt, and returns the apt id

deleteApartment(int id)
	*deletes apartment

deleteUser(int id)
	*deletes user

deleteNote(int id)
	*deletes note

login(String username,String password)
	*confirms if username/pass match


GET Methods:
createUser(String username,String password,String email)
	*creates user andreturns userid of new user

getUserInfo(String username,int userID)
	*display row in json format

getAllUsers(int apartmentID)
	*display all users where ApartmentID matches

getNotes(int apartmentID)
	*returns all notes listed under the apartment

addUserToApartment(int userID,int apartmentID)
	*adds user to apartment

getApartments(int apartmentID)
	*displays all users where ApartmentID matches

getMessages(int apartmentID)
	*displays all messages in json rows

getEvents(int apartmentID)
	*return all events in apartment as json rows
